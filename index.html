Features

Everything your team needs, in one place

From task boards to AI assistant to real-time huddles — Project Tracker is the operating system for your engineering team.

📋

Smart Task Management

Create, assign, and track tasks with priorities, stages, due dates, and progress tracking.

  • Custom stage workflows
  • Priority levels: Low, Medium, High, Critical
  • Assignee & due date tracking
  • Inline comments per task
📁

Project Workspaces

Organize work into projects with members, timelines, and progress visibility for everyone.

  • Project-level member access
  • Start & target date planning
  • Visual progress tracking
  • Color-coded project cards
✉️

Direct Messages

Private 1-to-1 messaging with unread badges, read receipts, and instant delivery.

  • Unread message counters
  • Read state tracking
  • One-click huddle from DM
📞

Huddle Voice Calls

Start instant voice huddles directly from the sidebar or DMs. Invite teammates, mute yourself, and get notified when someone joins your room — no third-party app required.

  • Instant room creation
  • Invite participants mid-call
  • Mute / unmute controls
  • In-app toast notifications when invited
🔔

Real-Time Notifications

Never miss an update — get notified for task assignments, status changes, comments, and reminders via in-app toasts and desktop push.

  • In-app toast stack
  • Desktop push (VAPID)
  • Email notifications via SMTP
  • Mark-all-read & clear
⏰

Task Reminders

Set reminders for any task with configurable lead times. Get notified before deadlines so nothing slips through the cracks.

  • Per-task reminder scheduling
  • Configurable advance notice
  • Reminder history view
🎫

Support Ticket System

Built-in ticketing module to track bugs, requests, and issues — separate from your project tasks for clean separation of concerns.

  • Project-linked tickets
  • Status & priority tracking
  • Assignee management
💬

Project Channels

Team messaging per project — keep conversations organized alongside the work they relate to.

  • Per-project chat channels
  • Sender avatars & timestamps
  • Unread message tracking
📎

File Attachments

Upload and attach files up to 150 MB to tasks or projects. All uploads stored securely and organized by workspace.

  • Up to 150 MB per file
  • Attach to tasks or projects
  • MIME type detection

Notifications

Stay in the loop, always

Multi-channel notifications via in-app toasts, desktop push, and email — so you never miss an important update whether you're active or away.

How it works

Up and running in minutes

No complex setup. Just create your workspace, invite your team, and start shipping.

1

Create Account

Sign up and choose to create a new workspace or join an existing one with an invite code.

2

Set Up Workspace

Name your workspace, configure email notifications, and add your AI API key for the assistant.

3

Invite Your Team

Share the auto-generated invite code. Team members can join instantly with their own accounts.

4

Start Shipping

Create projects, assign tasks, chat, huddle, and use the AI assistant to move fast together.
