Security

Built secure
from day one.

Security isn't a checkbox for Project Tracker — it's architecture. Every layer of the platform is designed to protect your team's data.

🔑

Google Authenticator 2FA

Time-based one-time passwords (TOTP) via Google Authenticator on every account. QR code enrollment with backup codes for recovery.

🛡️

bcrypt Password Hashing

All passwords are stored as salted bcrypt hashes. Even in the event of a data breach, raw passwords are never exposed.

🏢

Tenant Data Isolation

Every database query is workspace-scoped. No workspace can ever read, write or modify data belonging to another workspace.

👥

6 Role-Based Access Levels

Admin, Manager, TeamLead, Developer, Tester, and Viewer roles — each with precisely defined permissions on projects, tasks, members and settings.

🔒

HTTPS-Only Sessions

All traffic is encrypted in transit. Sessions are cookie-secured with server-side validation on every authenticated request.

🔑

Your Own AI Key

Your Anthropic API key is stored in your workspace settings only. Project Tracker never logs, shares or transmits it outside your own environment.
